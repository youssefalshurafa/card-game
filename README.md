# Hello World

Minimal Next.js 16 starter application using the App Router, JSX, and Tailwind CSS.

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run start
```

## Notes

- The homepage lives in app/page.jsx.
- The package name is card-game.
- If port 3000 is already occupied, the dev server will automatically move to the next available port.
